# 📸 Image Viewer OCR

一個類似 iPhone 圖片檢視器的網頁應用程式，支援影像處理、濾鏡效果和 OCR 文字辨識功能。

## ✨ 功能特色

### 📷 影像處理
- **裁切功能**：自由拖拉選擇要保留的區塊
- **尺寸調整**：自訂圖片大小

### 🎨 色彩調整（參考 iPhone 調整介面）
- 自動色階
- 曝光度
- 對比度
- 亮部 / 陰影
- 飽和度
- 色溫 / 色調
- 褐色調（懷舊效果）
- 清晰度

### 🖼️ 濾鏡效果
- 多種預設濾鏡（復古、黑白、暖色調等）
- 即時預覽

### 📝 OCR 文字辨識（核心功能）
- **智慧文字偵測**：自動偵測圖片中的文字區域
- **文字選取**：類似 iPhone 的操作方式
  - 長按啟動文字選取模式
  - 拖曳選擇文字範圍
  - 選取頭尾調整器
- **文字輸出**：
  - 複製到剪貼簿
  - 儲存為 TXT 文字檔

### 💾 檔案處理
- 選擇本地圖片檔案
- 拖放上傳
- 儲存處理後的圖片為新檔案

## 🛠️ 技術架構

- **前端框架**：HTML5 + CSS3 + JavaScript
- **影像處理**：Canvas API
- **OCR 引擎**：Tesseract.js
- **設計風格**：Nordic 銀白主題（延續專案設計語言）

## 📖 使用方式

### OCR 文字選取操作流程（電腦版）

1. **開啟圖片**：選擇或拖放圖片到應用程式
2. **啟動 OCR 模式**：點擊「文字辨識」按鈕
3. **等待辨識**：系統會自動偵測圖片中的文字
4. **選取文字**：
   - 方式一：點擊並拖曳選擇文字區域
   - 方式二：長按（500ms）進入精確選取模式，再調整頭尾
5. **複製或儲存**：使用工具列按鈕複製文字或儲存為檔案

## 🖥️ 版本規劃

本專案提供兩種使用方式：

### 🌐 Web 線上版本
- 部署於 GitHub Pages
- 無需安裝，直接透過瀏覽器使用
- 適合快速使用和跨平台存取

### 🍎 Mac 應用程式
- 使用 Electron 打包的原生應用程式
- 提供 DMG 安裝檔
- 更好的系統整合和效能

## 🚀 開始使用

### Web 版本
```bash
# 直接用瀏覽器開啟 index.html
open index.html

# 或使用本地伺服器
npx serve .
```

### Mac 版本
```bash
# 安裝依賴
npm install

# 開發模式
npm run electron:dev

# 打包 DMG
npm run electron:build
```
